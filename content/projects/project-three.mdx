# Project Three

An interactive Unity WebGL experience embedded directly in the portfolio gallery.
Approach the painting and click **▶ Play** to launch the game inline — no page navigation needed.

Replace the `shaderSrc` frontmatter field with the path to your exported Unity WebGL build folder (the one containing `index.html`).

## Tech Stack
- Unity 2023 LTS
- C# gameplay scripting
- WebGL export
